# Solavise Tech Women in Data — Cohort 2

## About the Programme

The Solavise Tech Women in Data Programme is a structured, community-driven learning experience designed to equip women with foundational and practical data skills. Cohort 2 runs from May to June 2026, covering Python for data analysis, tools like Pandas and NumPy, data visualisation, GitHub, and a collaborative hackathon to close out the programme.

This repository documents my journey through the programme — the assignments, the projects, and the growth.

---

## About Me

Hi, I am Ngumimi Bethel Tuse, a Computer Science student at Bingham University and an aspiring data engineer with a focus on the impact of healthcare data in Nigeria.

I have been building seriously for about six months. My current work includes a clinical heart disease ETL pipeline, an offline first aid CLI backed by a relational database, and a REST API built with Node.js and MongoDB. I also co-lead a 100+ member Women-in-Tech chapter at my university and interned at NITDA where I worked on Python automation and product design.

I believe that better data infrastructure leads to better healthcare outcomes — and I am working toward being part of the generation that builds it.

---

## My Expectations from the Programme

I joined Solavise because I need structure, community, and honest accountability. Self-directed learning is something I am capable of, but there is a difference between learning alone and learning alongside people who are building toward something real.

I expect this programme to challenge me technically and push me to be more consistent than I sometimes manage on my own. I also expect the community to be a space where I can ask questions without embarrassment and contribute meaningfully as I grow.

---

## What I Hope to Learn

- **Pandas in depth** — I have used it at a surface level, but I want to be confident with data cleaning, groupby operations, merging, and working with real messy datasets
- **Data visualisation** — turning numbers into something a non-technical person can understand and act on
- **Exploratory data analysis** — learning to ask the right questions of a dataset before building anything on top of it
- **NumPy fundamentals** — the layer underneath Pandas that I have mostly skipped
- **Collaborative working with Git** — I use Git for my own projects, but I want to get comfortable contributing to shared repositories

By the end of this programme, I want to have the skills to build the Nigeria Health Dashboard I have been planning — a project that visualises public health data for Nigerian communities. That is the goal sitting behind every session I attend.
