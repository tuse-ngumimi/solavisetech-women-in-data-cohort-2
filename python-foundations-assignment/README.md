##  How to Run

Make sure you have Python 3.x installed, then run any file from your terminal:

```bash
python data_types.py
python operators.py
python control_flow.py
python data_structures.py
```

Each file is self-contained and runs all exercises sequentially. Follow the prompts in your terminal.

---

##  A Note on Context

Where possible, examples are grounded in a Nigerian context — ₦ currency in the ATM simulation, African country capitals in the tuples exercise, and Naija-relevant names throughout. Data engineering is the lens; real-world relevance is the goal.
